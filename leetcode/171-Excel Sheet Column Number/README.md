# 171. Excel Sheet Column Number
## Easy

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:
```javascript
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
```
Example 1:

```javascript
Input: "A"
Output: 1
```
Example 2:

```javascript
Input: "AB"
Output: 28
```
Example 3:

```javascript
Input: "ZY"
Output: 701
```
