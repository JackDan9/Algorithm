# 168. Excel Sheet Column Title
## Easy

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:
```javascript
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...
```
Example 1:

```javascript
Input: 1
Output: "A"
```

Example 2:
```javascript
Input: 28
Output: "AB"
```
Example 3:
```javascript
Input: 701
Output: "ZY"
```
